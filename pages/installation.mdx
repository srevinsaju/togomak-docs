# Installation

togomak is distributed as a binary package. Since its open source, you may even compile it from source.

import { Tabs, Card, Cards, Callout } from 'nextra/components'

export let version = null;
import { useData } from 'nextra/data'
 
export const getStaticProps = ({ params }) => {
  return fetch(`https://api.github.com/repos/srevinsaju/togomak/releases/latest`)
    .then(res => res.json())
    .then(release => ({
      props: {
        // We add an `ssg` field to the page props,
        // which will be provided to the Nextra `useData` hook.
        ssg: {
          version: release.name
        }
      },
      // The page will be considered as stale and regenerated every 60 seconds.
      revalidate: 1500
    }))
}
 
export const Version = () => {
  // Get the data from SSG, and render it as a component.
  if (version === null) {
    version = useData().version
  }
  return version
}

<Tabs.Tab>

Download the right `togomak` version for your architecture and version, as defined by the environment variables
`VERSION` and `ARCH` in the following script for your machine.

```bash
export VERSION=1.3.0
export ARCH=amd64
wget https://github.com/srevinsaju/togomak/releases/download/v${VERSION}/togomak_${VERSION}_linux_${ARCH}.deb -O togomak.deb
sudo apt install ./togomak.deb
```
