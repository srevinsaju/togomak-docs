## Creating a new pipeline

```bash
togomak init
```

## Running a pipeline

```bash
togomak run
```
or 
```bash
togomak
```

## Running a pipeline with a specific stage

```bash
togomak stage_name
```

## Running multiple stages

```bash
togomak pipeline1 pipeline2
```
